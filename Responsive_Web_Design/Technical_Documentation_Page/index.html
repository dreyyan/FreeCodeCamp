Introduction

JavaScript is a programming language that allows you to implement complex features on web pages.

It powers dynamic content, interactivity, and is one of the core technologies of the web.

console.log("Hello, World!");
  • Client-side scripting
  • Event handling
  • DOM manipulation
Variables

Variables are containers for storing data values.

JavaScript uses `var`, `let`, and `const` for variable declarations.

let name = "Ada"; const PI = 3.14159;
  • var: function scoped
  • let: block scoped
  • const: block scoped, cannot be reassigned
Functions

Functions are blocks of code designed to perform a particular task.

They are executed when invoked (called).

function greet(name) {
  return "Hello " + name;
}
  • Reusable
  • Can accept parameters
  • Can return values
Loops

Loops are used to repeatedly run a block of code while a condition is true.

Common loop types include `for`, `while`, and `do...while`.

for (let i = 0; i < 5; i++) {
  console.log(i);
}
  • For loop
  • While loop
  • Do-while loop
Objects

Objects are collections of key-value pairs in JavaScript.

They can contain properties and methods.

const person = {
  name: "Ada",
  age: 36
};
  • Store related data
  • Can contain functions (methods)
  • Essential to OOP in JavaScript
